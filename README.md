# parcel-typescript-react-example

A minimum TypeScript-react app with Parcel Bundler.

## Source code

* [index.html](index.html)
* [index.tsx](src/index.tsc)
* [styles.css](src/styles.css)

For more details, see src/.

## Usage

### Installation

> yarn

### Development

Launch development server.

> yarn start
